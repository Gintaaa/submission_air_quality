# Air Quality Dashboard ☁️

## Analisis Utama
* **Stasiun**: Dongsi memiliki polusi PM2.5 tertinggi, sedangkan Dingling terendah.
* **Tren**: Kualitas udara membaik secara signifikan pada bulan Mei hingga Agustus.
* **Musim**: Polusi udara mencapai puncaknya pada musim Winter.

## Cara Menjalankan
1. Install library:
   ```bash
    pip install pandas matplotlib seaborn streamlit

2. Jalankan dashboard:
    ```bash
    streamlit run dashboard/dashboard.py
